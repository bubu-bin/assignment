## General info
With this app you can purchase and search for items with dynamically created forms
	
## Technologies
Project is created with:
* Node: 14.18.2
* React: 18.2.0
* Prisma: 4.14.1
* Typescript: 5.0.4
* PostgreSQL: 14.6
	
## Setup
To run this project, follow these steps:

```
$ git clone https://github.com/bubu-bin/assignment.git
```

```
$ cd assignment
```

```
$ docker-compose up
```
